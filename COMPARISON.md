# Comparison

| Library                                                                      | Description                                                     |
| ---------------------------------------------------------------------------- | --------------------------------------------------------------- |
| [`chess.js`](https://www.npmjs.com/package/chess.js)                         | full chess engine — FEN parsing bundled, not standalone         |
| [`chessops`](https://www.npmjs.com/package/chessops)                         | TypeScript chess library — FEN support included, not standalone |
| [`chess-fen`](https://www.npmjs.com/package/chess-fen)                       | standalone immutable FEN manipulation                           |
| [`@chess-fu/fen-parser`](https://www.npmjs.com/package/@chess-fu/fen-parser) | lightweight regex-based FEN parser                              |
